# Dijkstra Routing Visualizer

An interactive visualization tool for understanding how Dijkstra's Routing Algorithm works in computer networks.

## Features

- Add routers dynamically
- Connect routers with weighted links
- Select source and destination routers
- Visualize shortest path calculation
- Step-by-step algorithm execution
- Interactive network topology creation

## Tech Stack

- React
- TypeScript
- Lucide React

## Installation

```bash
npm install
npm start
